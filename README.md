# Siruya

Siruya는 콩나물재배기 같은 소형 식물 재배 기기를 스마트폰에서 등록하고 제어하기 위한 IoT 앱입니다.

## 주요 기능

- 기기 등록: 기기가 만든 `WaterPlant-*` Wi-Fi에 연결한 뒤 집 Wi-Fi 정보를 전달합니다.
- 기기 목록: 등록된 재배기의 연결 상태와 물/팬 동작 상태를 확인합니다.
- 기기 제어: 전원, 물 공급, 팬, 청소 모드를 앱에서 제어합니다.
- 상태 확인: 기기 상태를 주기적으로 읽고 온라인/오프라인 상태를 표시합니다.
- 펌웨어 업데이트: 기기 프로그램의 최신 버전을 확인하고 업데이트 명령을 보냅니다.

## 프로젝트 구성

- `screens/`: 앱 화면
- `features/devices/`: 기기 등록, 상태 확인, 제어 기능
- `firmware/`: 콩나물재배기 내부에 들어가는 기기 프로그램
- `android/`, `ios/`: Android와 iOS 앱 실행 설정

## 실행

```bash
npm install
npm start
npm run android
```

현재 코드는 앱 첫 화면에서 바로 기기 관리 화면을 보여줍니다.

## PC에서 한 번에 실행

Windows에서는 프로젝트 폴더의 `Siruya_Run_Android.bat` 파일을 더블클릭하면 됩니다.

이 파일은 다음 작업을 자동으로 진행합니다.

- Android Studio에서 `android` 프로젝트 열기
- Android 가상폰 실행
- React Native 개발 서버 실행
- 가상폰에 앱 설치 및 실행

가상폰이 여러 개 있으면 기본적으로 `Pixel_7`을 우선 사용합니다. 다른 가상폰을 쓰려면 Windows 환경 변수 `SIRUYA_AVD`에 가상폰 이름을 넣으면 됩니다.

## 에뮬레이터 화면 확인용 데모

에뮬레이터에서는 실제 기기 Wi-Fi에 연결할 수 없으므로, 개발 중에는 앱의 `+` 버튼을 누른 뒤 `데모 연결 흐름 보기`를 선택하면 됩니다.

이 데모는 실제 기계 없이 다음 화면을 확인하기 위한 기능입니다.

- 기기 Wi-Fi 연결 안내
- 집 Wi-Fi 선택
- 장치 이름 설정
- 연결 후 상세 제어 화면
- 홈의 기기 카드

데모로 추가된 기기는 `화면 확인용`으로 표시되며, 실제 기계에는 명령을 보내지 않습니다.
